# Premium Booking SaaS

Production-ready structure blueprint for a premium mobile-first booking SaaS.

This repository is intentionally not a full generated project yet. The first layer defines the architecture, routing, UI system, design tokens, layout rules, and deployment boundaries so the MVP can be built without early structural debt.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- Railway
